# ESP32 + JSN-SR04T → ESP RainMaker
## Датчик уровня жидкости

---

## 📦 Необходимые компоненты

| Компонент | Кол-во |
|-----------|--------|
| ESP32 (любая плата: DevKit, WROOM и т.д.) | 1 |
| Датчик расстояния JSN-SR04T | 1 |
| Резистор 1 кОм | 1 |
| Резистор 2 кОм | 1 |
| Провода, макетная плата | — |

---

## 🔌 Схема подключения

```
JSN-SR04T          ESP32
─────────          ─────
  VCC  ──────────► 5V (или Vin)
  GND  ──────────► GND
  TRIG ──────────► GPIO 5
  ECHO ──┬──[1kΩ]──► GPIO 18
         └──[2kΩ]──► GND
```

> ⚠️ **ВАЖНО:** ECHO выдаёт 5 В, а GPIO ESP32 рассчитан на 3.3 В.
> Делитель напряжения (1 кОм + 2 кОм) понижает сигнал до ~3.3 В.
> Без делителя GPIO может сгореть!

## 🛠 Установка библиотек (Arduino IDE)

1. Открой **Arduino IDE → Tools → Manage Libraries**
2. Найди и установи:
   - `ESP RainMaker` от Espressif Systems
3. В **Tools → Board** выбери: `ESP32 Dev Module` (или свою плату)
4. Выбери правильный **COM-порт**

## ⚙️ Калибровка резервуара

Открой файл `.ino` и отредактируй эти константы под свой бак:

```cpp
// Расстояние от датчика до дна при ПУСТОМ резервуаре (см)
#define TANK_EMPTY_CM   100.0

// Расстояние от датчика до поверхности при ПОЛНОМ резервуаре (см)
#define TANK_FULL_CM     10.0

// Порог «низкого уровня» для алерта (%)
#define LOW_LEVEL_THRESHOLD  20
```

**Как измерить:**
1. Опусти бак. Измерь расстояние рулеткой от датчика до дна → `TANK_EMPTY_CM`
2. Налей полный бак. Измерь расстояние от датчика до воды → `TANK_FULL_CM`

## 📱 Первый запуск и настройка RainMaker

1. **Прошей** ESP32 (Upload в Arduino IDE)
2. Установи приложение **ESP RainMaker** (iOS / Android)
3. В приложении нажми **"Add Device"** → **"BLE Provisioning"**
4. Найди устройство с именем `PROV_WATER_LEVEL`
5. Введи свои данные Wi-Fi и POP-код: **`abcd1234`**
   *(можно изменить в коде: строка `"abcd1234"`)*
6. После подключения устройство появится в приложении как **"Water Level Monitor"**

## 📊 Параметры в приложении

| Параметр | Описание |
|----------|----------|
| **Level** | Уровень жидкости в % (0–100) |
| **LowAlert** | true/false — низкий уровень |

---

## 🔔 Настройка уведомлений в RainMaker

1. В приложении зайди в устройство → **Automations**
2. Создай автоматизацию: когда `LowAlert = true` → отправить уведомление
3. Укажи желаемое сообщение (например, "Уровень воды низкий!")

---

## 🔄 Обновление прошивки (OTA)

OTA включён автоматически. Для обновления:
- Зайди в приложение RainMaker → устройство → **OTA Update**

## ❓ Частые проблемы

| Симптом | Решение |
|---------|---------|
| Всегда 0% или 100% | Проверь `TANK_EMPTY_CM` и `TANK_FULL_CM` |
| "Таймаут — нет отклика" | Проверь питание датчика (нужен 5V) и делитель напряжения |
| Устройство не появляется в BLE | Нажми Reset на ESP32, повтори провижининг |
| Wi-Fi не подключается | Убедись что сеть 2.4 ГГц (5 ГГц не поддерживается) |
